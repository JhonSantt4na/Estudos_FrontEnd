    Ajustes de Textos = hr, br, & n b s p

    hr = Divide com uma linha
    br = Pula uma linha
    & n b s p (TudoJunto) = Dar espaços em brancos

    Demais TAGS = b, i, u, small, strong, del, sub, sup, mark

    b = Negrito
    i = Itálico
    u = Sublinhado
    small = Deixa pequeno
    strong = Ênfase
    del = = Risco no meio
    sub = Texto embaixo
    sup = Texto elevado
    mark = Marca texto

    Elementos de Citação = abbr, address, cite, q, blockquote, bdo rtl e ltr

    AVON

    address = Algum endereço ou link

    cite = Texto em itálico
    q = Coloca entre aspas
    blockquote = Da uma previa do texto

    rtl = Direita para esquerda
    ltr = Esquerda para Direita
